Two-Agent Embed Example

This demonstrates how to embed the two-agent chat interface in your website

1. Basic Usage

// URL 파라미터 구성 const params = new URLSearchParams({ topStatement: "여성은 가정에서 아이를 돌봐야 한다고 생각합니다", aiSummary: "전통적인 성별 역할에 대한 견해", importanceLevel: "high", userReasoning: "가족의 안정성을 위해서는 전통적인 역할 분담이 필요하다고 생각합니다" }); // 임베드 URL 생성 (개발 환경용) const embedUrl = 'http://localhost:5173/two-agent-embed?' + params.toString(); // iframe으로 임베드 const iframe = document.createElement('iframe'); iframe.src = embedUrl; iframe.width = '100%'; iframe.height = '600px'; iframe.frameBorder = '0'; document.body.appendChild(iframe);

4. Conversation Flow

1. Initial Message (AI Assistant) ↓ 2. User Message ↓ 3. Emotional Agent Response (💝) ↓ 4. Intelligent Agent Response (🧠) ↓ 5. User Message ↓ 6. Repeat steps 3-5...

5. Features

  • Dual Agent System: Each user message gets responses from both emotional and intelligent agents
  • Visual Distinction: Different styling and icons for each agent type
  • Personalized Initial Message: Based on URL parameters
  • Responsive Design: Works on desktop and mobile
  • Clean Interface: No branding or external elements
